# Contribuindo / Contributing

🇧🇷 **Português**

Obrigado por querer contribuir! O guia completo de contribuição — como rodar,
padrões de documentação (bilíngue, estilo tiangolo), padrões de código e commits
— está na documentação:

➡️ **<https://mauriciobenjamin700.github.io/django-survival-guide/sobre/contribuindo/>**

Resumo rápido:

```bash
uv sync --all-groups
make test          # testes precisam passar
make docs-build    # mkdocs build --strict precisa sair limpo
```

Commits seguem [Conventional Commits](https://www.conventionalcommits.org/)
(`feat`, `fix`, `docs`, `test`, `refactor`, `chore`, `ci`).

---

🇺🇸 **English**

Thanks for wanting to contribute! The full contribution guide — how to run,
documentation standards (bilingual, tiangolo style), code standards and
commits — lives in the docs:

➡️ **<https://mauriciobenjamin700.github.io/django-survival-guide/en/sobre/contribuindo/>**

Quick summary:

```bash
uv sync --all-groups
make test          # tests must pass
make docs-build    # mkdocs build --strict must come out clean
```

Commits follow [Conventional Commits](https://www.conventionalcommits.org/)
(`feat`, `fix`, `docs`, `test`, `refactor`, `chore`, `ci`).
